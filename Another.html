What you've seen first when you open the website is the use of JavaScript.

**JavaScript** is a high-level, interpreted programming language commonly used to create interactive effects and dynamic behavior on websites.
It is a core technology of the web, alongside HTML and CSS, and enables developers to build responsive user interfaces, control multimedia, animate elements, and handle events in real time.
JavaScript can run in web browsers and on servers (using environments like Node.js).
