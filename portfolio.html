2. CSS: The Presentation

CSS (Cascading Style Sheets) is used to control the presentation of that content. It styles the HTML by setting colors, fonts, spacing, and overall layout. While HTML establishes what the content is, CSS defines how it should look, ensuring that the webpage is visually engaging and user-friendly.
